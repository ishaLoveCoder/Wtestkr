# TamilMV + TamilBlasters Torrent Bot

A fully automated Telegram torrent posting bot built using Telebot.

This bot automatically scrapes:

- TamilMV
- TamilBlasters

and posts new `.torrent` files directly into your Telegram channel.

---

# Features

✅ TamilMV Auto Scraper

✅ TamilBlasters Auto Scraper

✅ Auto Torrent Posting

✅ Duplicate Post Protection

✅ JSON Database Support

✅ 15 Minute Auto Check

✅ Render Deployment Support

✅ Docker Support

✅ Custom Thumbnail Support

✅ BOT_TAG Support

✅ Flask Health Server
